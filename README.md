# react-ts-handson

React + TypeScript + Vite の練習用ワークスペースです。

## フォルダ構成

- `react-ts-handson/`
  - カウンターアプリのサンプル
- `todo-app/`
  - Todo アプリのサンプル

各プロジェクトはそれぞれ独立した Vite アプリです。

## よく使うコマンド

各プロジェクトのディレクトリに移動して実行します。

```bash
npm install
npm run dev
npm run build
```

## 補足

- `src/` に React コンポーネントや CSS があります
- `public/` に静的ファイルがあります
- `dist/` はビルド結果です
